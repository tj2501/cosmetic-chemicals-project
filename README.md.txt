# Cosmetic Chemicals Analysis
## Analyze FDA cosmetic data to find safety risks (e.g., missing CAS numbers).

## How to Run
1. Open `notebooks/analysis.ipynb` in Jupyter.
2. Run all cells.

## Data Source
https://safecosmetics.cdph.ca.gov/search/
http://www.cdph.ca.gov/programs/cosmetics/Pages/default.aspx
